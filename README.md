# Browser BI Tool

ブラウザ完結型のビジネスインテリジェンスツール。サーバー不要でCSV/Excelファイルを読み込み、豊富なチャートで可視化、PowerPoint/Excelへエクスポートできます。

## 特徴

- **ゼロインストール**: ブラウザのみで動作、インストール不要
- **プライバシー保護**: データは外部サーバーに送信されません
- **豊富なチャート**: 棒グラフ、折れ線、円グラフ、組合せチャート等
- **フィルター・スライサー**: インタラクティブなデータ絞り込み
- **エクスポート**: PowerPoint、Excel、PNG、SVG形式で出力

## 技術スタック

- **フレームワーク**: Vue.js 3 + Vite
- **状態管理**: Pinia
- **チャート**: ECharts
- **エクスポート**: SheetJS (xlsx), pptxgenjs
- **スタイル**: TailwindCSS

## 開発環境セットアップ

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# 本番ビルド
npm run build

# プレビュー
npm run preview
```

## 機能一覧

### データインポート
- CSV ファイル読み込み
- Excel (.xlsx/.xls) ファイル読み込み
- JSON ファイル読み込み
- データ型自動検出
- シート選択（Excel）

### チャート
- 基本チャート: 棒グラフ、折れ線、円グラフ、散布図、面グラフ
- 高度なチャート: 組合せチャート、レーダー、ヒートマップ、ツリーマップ、サンキー図
- インタラクティブ: ズーム、ツールチップ、凡例操作

### フィルター・スライサー
- ドロップダウンフィルター
- 範囲スライサー（数値）
- 日付スライサー
- マルチセレクト
- フィルター連動（複数チャート同時適用）

### エクスポート
- PNG 画像出力
- SVG ベクター出力
- Excel ファイル出力
- PowerPoint スライド出力

### ダッシュボード
- 複数チャート配置
- グリッドレイアウト（ドラッグ&ドロップ）
- 設定保存・復元（ローカルストレージ）
- 設定共有（JSON形式）
