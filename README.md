# Browser BI Tool

ブラウザ完結型のビジネスインテリジェンスツール。サーバー不要でCSV/Excelファイルを読み込み、豊富なチャートで可視化、PowerPoint/Excelへエクスポートできます。

## デモ

**GitHub Pages**: https://chel2311.github.io/BI-Tool/

## 使い方

### オンライン版
上記のGitHub Pagesにアクセスするだけで利用可能です。

### ローカル版（オフライン利用）
1. `release` フォルダをダウンロード
2. `start.bat` をダブルクリック
3. ブラウザが自動で開きます（http://localhost:8080）
4. 終了時はコマンドプロンプトを閉じる

## 特徴

- **ゼロインストール**: ブラウザのみで動作
- **プライバシー保護**: データは外部サーバーに送信されません
- **豊富なチャート**: 棒グラフ、折れ線、円グラフ、組合せチャート、散布図、面グラフ
- **柔軟な集計**: 合計、平均、カウント、最大、最小
- **フィルター機能**: 条件による絞り込み、複数条件対応
- **日付単位変換**: 年/四半期/月/週/日でグルーピング
- **エクスポート**: PNG画像、PowerPoint、Excel、CSV形式で出力

## 機能一覧

### データインポート
- CSV ファイル読み込み
- Excel (.xlsx/.xls) ファイル読み込み
- ドラッグ&ドロップ対応
- 文字コード自動判定
- サンプルデータ内蔵

### チャート種別
| チャート | 説明 |
|---------|------|
| 棒グラフ | カテゴリ別比較、積み上げ対応 |
| 折れ線グラフ | 時系列推移 |
| 円グラフ | 構成比率 |
| 組合せチャート | 棒+折れ線（2軸）、別々の集計方法 |
| 散布図 | 2変数の相関 |
| 面グラフ | 累積推移 |

### 集計・分析機能
- 集計方法: 合計 / 平均 / カウント / 最大 / 最小
- グループ化（系列分割）
- 積み上げモード（通常/100%）
- ソート（値順/ラベル順）
- ズーム機能

### フィルター
- 等しい / 等しくない
- 含む / 含まない
- 以上 / 以下 / 範囲指定
- 空 / 空でない
- 複数条件のAND/OR

### 日付単位変換
日付カラムを以下の単位に集約:
- 年（2024）
- 四半期（2024-Q1）
- 月（2024-01）
- 週（2024-W01）
- 日（2024-01-01）

### エクスポート
- PNG画像として保存
- PowerPointスライドに出力
- 集計データをExcel出力
- フィルタ済みデータをCSV出力

### ダッシュボード
- 複数チャートを1画面に配置
- チャート/テーブルパネル対応
- 設定の保存・復元

### テーブル分析
- データの詳細表示
- ソート・フィルター機能

## 技術スタック

- **フレームワーク**: Vue.js 3 + Vite
- **状態管理**: Pinia
- **チャート**: ECharts
- **スタイル**: TailwindCSS
- **Excel処理**: SheetJS (xlsx)
- **PowerPoint出力**: pptxgenjs

## ディレクトリ構成

```
browser-bi-tool/
├── release/              # 配布用（ユーザーはここだけ使用）
│   ├── start.bat         # 起動スクリプト
│   ├── server.ps1        # ローカルサーバー
│   └── BI-Tool/          # アプリ本体
│
├── src/                  # ソースコード（開発用）
│   ├── components/       # UIコンポーネント
│   ├── services/         # ビジネスロジック
│   ├── stores/           # Pinia ストア
│   ├── views/            # ページコンポーネント
│   └── router/           # Vue Router
│
├── dist/                 # ビルド出力
├── public/               # 静的ファイル
└── docs/                 # ドキュメント
```

## 開発

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# 本番ビルド
npm run build

# GitHub Pagesにデプロイ
npm run deploy
```

### ビルド後のrelease更新

```bash
npm run build
cp -r dist/* release/BI-Tool/
```
